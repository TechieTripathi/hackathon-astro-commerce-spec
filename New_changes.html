Backend Development Report

Backend New Features

New Changes

  • Real-Time Customer Support System

    a Real-Time Customer Support System using WebSockets, enabling customers to communicate directly with support agents regarding orders, payments, product issues, and delivery tracking.

  • EMI Payment Option

    Using the Razorpay API, customers can choose EMI plans during checkout. All installment payments are managed by the respective bank. The system receives payment confirmations and notifications from Razorpay to keep order and payment records updated.

  • Product Tracking Feature

    Product Tracking Feature: There are two approaches to implement product tracking. The first approach is to maintain an Order Tracking table containing fields such as order_id, user_id, and status. The order status is updated at different stages of the delivery process, such as Pending, Packed, Shipped, Out for Delivery, and Delivered. Customers can view the latest status of their orders in real time. The second approach is to integrate with third-party logistics providers such as Delhivery and Google Maps API can be integrated to provide real-time shipment and location updates.

Backend API Section

  • Product Image API

    Supports multiple images for a single product. Users can view different product angles and variants.

  • Live Location API (Google Maps API)

    Provides real-time location tracking for deliveries and orders. Integrated with Google Maps for accurate navigation and tracking.

System Flows

User → Browse Products → Add to Cart → Checkout → Payment → Order Confirmation → Product Tracking
Admin → Manage Inventory → Update Product Details → Upload Product Images → Monitor Orders
Customer → Raise Support Request → Real-Time Customer Support → Resolution
